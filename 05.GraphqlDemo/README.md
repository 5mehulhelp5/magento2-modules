# Pankaj_GraphqlDemo

## Purpose
This module is created to demonstrate how to build custom GraphQL queries in Magento 2.

## Goal
- Define custom GraphQL schema and queries
- Fetch data using custom resolvers
- Follow Magento 2 GraphQL best practices
- Provide clean and reusable GraphQL APIs

## Use Case
- Fetch greeting messages via GraphQL
- Retrieve product details using SKU
- Use in headless Magento or PWA integrations
- Demonstrate GraphQL implementation in Magento 2 interviews
