# Pankaj_GraphqlAuth

## Purpose
This module is created to demonstrate authenticated GraphQL queries in Magento 2.

## Goal
- Create secured GraphQL queries using authentication
- Restrict API access to logged-in users
- Use custom resolvers with Magento GraphQL
- Follow Magento 2 GraphQL security best practices

## Use Case
- Fetch product details only for authorized users
- Secure sensitive data in headless Magento setups
- Integrate Magento GraphQL with authenticated frontend apps
- Demonstrate GraphQL authentication in Magento 2 interviews
